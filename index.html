About Me

I am a PhD student in the Department of Electrical and Computer Engineering at Stony Brook University working with Professor Dantong Yu. I am interested in high performance networks and storage systems. Precisely, I am working on an RDMA-based FTP data transfer software - RFTP and a NUMA-aware Cache for Linux SCSI Target Framework with iSCSI/iSER drivers.

Before came to Stony Brook, I worked as a senior software engineer for an online transaction processing system in China UnionPay for three years, and as a software tester (intern) in Samsung Electronics (China) R&D Center for 11 months. I received the B.S. degree from Wuhan University, China, in 2005 and the M.E. degree from Nanjing University, China, in 2007, both in Software Engineering.

Software

RFTP: RDMA-based FTP Service

RFTP is an RDMA-based data transfer software for high performance networks, such as InfiniBand, RoCE, and iWARP. In the meantime, Linux TCP zero-copy interfaces, sendfile and splice, are also integrated into RFTP to accommodate network environments without RDMA supports. It also integrates an asynchronous I/O module for efficient disk access. The software architecture of RFTP is a combination of event-driven and multi-threaded programming model to maximize network and disk bandwidth utilization and minimize CPU utilization.


rperf: Benchmark RDMA Network Performance as iperf

rperf is an RDMA compatible version of iperf. It takes advantage of original iperf software architecture, and adds several RDMA extentions into iperf. rperf features several parameters to achieve the best performance of RDMA hardware.


Open Source Contributions

I contributed an RDMA I/O engine and a NUMA binding module for FIO - Flexible I/O Tester.
